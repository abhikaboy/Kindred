## Summary

### Key Principles

1. **Cache for Performance** - Avoid unnecessary API calls
2. **Invalidate on Mutation** - Prevent stale data
3. **Optimistic Updates** - Instant UI feedback
4. **Fetch After Invalidation** - Ensure consistency
